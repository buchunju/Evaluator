### A brainfuck evaluator in javascript.

---

A script to evaluate brainfuck code.

Overview:

- Evaluating small code.
- Doesn't support input yet

Simple hello world code.

```
>>++++[<+++++[<+++++>-]>-]<<++++.---.>++[<+++>-]<+..+++.>>>++++[<++++[<++>-]>-]<<.--[>++++<-]>-.>++++[<-->-]<.+++.>+++[<-->-]<.>++++[<-->-]<.
```

Modify the code variable within the _brainf.js_ file.

Run with node.

```sh
node  brainf.js
```
